HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto). É a linguagem usada para criar páginas da web.

Exemplo de Código HTML:

!DOCTYPE html
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>Minha Primeira Página</title>
  </head>
  <body>
    <h1>Olá, mundo!</h1>
    <p>Este é o meu primeiro site.</p>
  </body>
</html>
